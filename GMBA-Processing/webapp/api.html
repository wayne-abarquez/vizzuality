The web services available are:

  • taxonId: the id of the taxon in the GBIF backbone (nub) taxonomy. This is mandatory
  • minLatitude: the minimum latitude
  • minLongitude: the minimum longitude
  • minElevation: the minimum elevation in meters
  • maxElevation: the maximum elevation in meters
  • minRelief: the minimum relief
  • maxRelief: the maximum relief
  • tvzCode: the tvzCodes (see WIKI). This can be repeated to logically OR tvzCodes
  • Example 1: /service/search?taxonId=14706247
  • Example 2: /service/search?taxonId=14706247&minElevation=-4&maxElev....

  • taxonId: the id of the taxon in the GBIF backbone (nub) taxonomy. This is mandatory
  • rank: one of kingdom,phylum,class,order,family,genus,species(will include subspecies) or nub(will not include subspecies). This is mandatory
  • minLatitude: the minimum latitude
  • maxLatitude: the maximim latitude
  • minLongitude: the minimum longitude
  • maxLongitude: the maximim longitude
  • minElevation: the minimum elevation in meters
  • maxElevation: the maximum elevation in meters
  • minRelief: the minimum relief
  • maxRelief: the maximum relief
  • tvzCode: the tvzCodes (see WIKI). This can be repeated to logically OR tvzCodes
  • Example 1: /service/download?taxonId=14706247&rank=species
